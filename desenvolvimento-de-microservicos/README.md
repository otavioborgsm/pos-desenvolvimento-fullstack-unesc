Disciplina de Desenvolvimento de Microserviços onde foi desenvolvido uma aplicação para restaurantes utilizando-se deste conceito.
>
#### Pontos a se destacar do projeto:
 * Rabbit MQ como Message Broker;
 * API Gateway para controle das rotas e chamadas;
 * Cada microserviço com seu próprio banco de dados.
